# cluster_watch: Live Monitoring of a Supercomputer

## Purpose
This `Python` tool provides live graphical monitoring of a cluster (of e.g. hybird CPU and GPU nodes), by providing a list of hostnames.

## Dependencies
* Python/3.6-intel-2018a
* Python Tkinter
